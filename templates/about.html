Bridging the Gap Between Citizens and Welfare

SchemeAI is a modern, AI-powered system designed to remove complex administration barriers, helping you discover eligible central and state government schemes in seconds.

Our Mission

To empower every Indian citizen, especially in rural and underserved areas, with immediate knowledge of government welfare schemes. We aim to replace manual research with AI matching, ensuring no eligible individual misses critical support.

Our Vision

To build a transparent, decentralized, and intelligent citizen portal. By leveraging advanced machine learning classifications, we envision a future where welfare application processes are automated, intuitive, and accessible to everyone.

System Workflow

1. User Input ingestion

User inputs profile criteria (age, state, income, caste, disability, occupation) via the multi-step recommendation wizard.

2. ML Pipeline Processing

The backend scales inputs and feeds the categorical parameters into the Random Forest Multi-Output Classifier to score eligible probabilities.

3. Dynamic Filtering

Hard constraints (like strict age limits or specific state locations) filter the candidate schemes to enforce 100% correct criteria bounds.

4. Report & Actions

Eligible schemes are listed with direct application links, document checklists, and downloadable PDF reports.

Technology Stack

  • Frontend: HTML5, CSS3 (Custom Design System), JavaScript (ES6), Bootstrap 5, Font Awesome, Poppins Font
  • Backend: Python, Flask Micro-Framework
  • Database: SQLite3 (structured for relational scale)
  • Machine Learning: Scikit-learn (Random Forest Multi-Output Classifier), Pandas, NumPy, Joblib
  • PDF Engine: HTML5 Canvas rendering & html2pdf.js

Key Advantages

  • Seconds Search: Reduces scheme search times from hours of reading websites to a 20-second form.
  • Accurate Eligibility: Combining ML scores with hard rule filters ensures users see only real, valid matches.
  • Structured Comparisons: Side-by-side matrices prevent users from applying to inferior/conflicting schemes.
  • Inclusive Features: Integrates Voice Search and Multi-language support (Hindi, Marathi) for rural accessibility.

Future Scope

  • Automatic Form Auto-Fill: Deep integration to populate application forms from the citizen's secure dashboard vault.
  • Government API Sync: Connect directly to central/state welfare APIs for real-time application tracking.
  • Advanced LLM Chatbot: Upgrade the local keyword bot to a generative Large Language Model for human-like query support.
  • Document OCR Verification: Scan documents (like caste/income certificates) using OCR to verify profile eligibility values automatically.
